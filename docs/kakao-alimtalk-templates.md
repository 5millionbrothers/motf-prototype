# moTF 카카오 알림톡 템플릿 초안

작성일: 2026-07-05

## 기준

- 알림톡은 정보성 메시지만 발송한다.
- 광고, 혜택, 이벤트, 홍보성 문구는 넣지 않는다.
- 템플릿 문구가 승인된 뒤 문장 구조를 바꾸면 재심사가 필요할 수 있다.
- 숙소명, 예약번호, 금액, 날짜, 링크 등은 변수로 둔다.
- 채팅 알림은 같은 대화방 기준 5분 안에는 1회만 발송한다.
- 관리자 알림에서는 새 회원가입, 입금 완료 예약 알림은 제외한다.
- 알림톡 본문에 긴 URL을 직접 넣지 않고, 웹링크 버튼으로 상세 화면 또는 처리 화면으로 이동시킨다.

## 공통 버튼 후보

| 버튼명 | 용도 |
| --- | --- |
| 예약내역 보기 | 이용자 마이페이지 예약 상세 |
| 주문내역 보기 | 이용자 마이페이지 주문 상세 |
| 채팅 확인 | 이용자/사장님 채팅방 |
| 관리자 확인 | 관리자 상세 화면 |
| 정산 확인 | 사장님/관리자 정산 화면 |
| 수락 처리 | 사장님 예약/주문 수락 전용 화면 |
| 취소 처리 | 사장님 예약/주문 취소 전용 화면 |

## 버튼 링크 전략

- 알림톡 버튼은 웹링크(WL) 버튼을 기본으로 사용한다.
- 이용자 버튼은 `https://motf.co.kr/...` 주소로 연결한다.
- 사장님 버튼은 `https://motfowner.co.kr/...` 주소로 연결한다.
- 관리자 버튼은 관리자 권한 화면으로 연결한다.
- 수락/취소 버튼은 바로 처리하지 않고, 전용 확인 화면으로 이동시킨 뒤 한 번 더 확인 버튼을 누르게 한다.
- 수락/취소 링크에는 짧은 일회용 토큰을 포함한다.
- 일회용 토큰은 특정 예약/주문, 특정 사장님, 만료 시간, 처리 권한을 서버에서 검증한다.
- 이미 처리된 건을 다시 누르면 현재 상태만 안내한다.
- URL은 템플릿 본문이 아니라 버튼 URL 변수로 관리한다.
