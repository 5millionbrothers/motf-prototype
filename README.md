# moTF 이용자 프로토타입

대학생 MT 팀을 위한 숙소 예약, 공판장 주문, 채팅, 리뷰, 마이페이지 흐름을 확인하는 정적 웹 프로토타입입니다.

## GitHub Pages 배포

이 저장소를 public으로 만들고 GitHub Pages를 활성화하면 `index.html`이 공개 웹사이트로 배포됩니다.

권장 설정:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

## 프로토타입 범위

현재 버전은 실제 API 없이 브라우저에서 동작합니다. 결제, 지도, 회원, 실시간 채팅, 데이터 저장은 추후 개발 단계에서 연결해야 합니다.
